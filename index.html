Kebijakan Privasi

Terakhir diperbarui: 1 Desember 2025

1. Pendahuluan

KKSA-Ku ("Aplikasi", "Kami") menghormati privasi pengguna dan berkomitmen untuk melindungi data pribadi yang dikumpulkan melalui aplikasi. Kebijakan ini menjelaskan jenis data yang dikumpulkan, bagaimana data digunakan, serta hak pengguna terkait data tersebut.

2. Penggunaan Kamera

Aplikasi KKSA-Ku menggunakan kamera perangkat hanya untuk tujuan berikut:

  • Scan QR Code: Kamera digunakan untuk memindai QR code yang diperlukan untuk verifikasi atau proses layanan koperasi.
  • Unggah Foto Profil: Pengguna dapat mengambil foto menggunakan kamera untuk mengunggah foto profil akun.

Kami tidak menyimpan atau memproses data kamera kecuali foto yang diunggah secara sadar oleh pengguna. Kamera tidak digunakan untuk pengenalan wajah (face recognition), pemantauan latar belakang, atau tujuan tidak sah lainnya.

3. Informasi yang Kami Kumpulkan

Aplikasi dapat mengumpulkan informasi berikut:

  • Foto profil yang diunggah pengguna
  • Hasil pemindaian QR code

Tidak ada data kamera tambahan yang dikumpulkan tanpa persetujuan.

4. Penyimpanan dan Keamanan Data

Data yang dikumpulkan disimpan di server internal koperasi sesuai kebutuhan layanan. Kami menerapkan langkah-langkah keamanan yang wajar untuk melindungi data pengguna. Data tidak dibagikan ke pihak ketiga tanpa persetujuan pengguna, kecuali jika diwajibkan oleh hukum.

5. Izin yang Digunakan

Aplikasi membutuhkan izin berikut:

  • android.permission.CAMERA — Digunakan untuk memindai QR code dan mengambil foto profil. Aplikasi tidak mengakses kamera tanpa tindakan eksplisit dari pengguna.

6. Hak Pengguna

Pengguna memiliki hak untuk:

  • Mengubah atau menghapus foto profil
  • Meminta perubahan data akun (jika fitur tersedia)
  • Menghubungi kami untuk pertanyaan terkait privasi
